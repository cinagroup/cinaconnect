# ERC-6492 API

> `@cinacoin/erc6492` — ERC-6492 签名验证（合约钱包签名）。

## Installation

```bash
npm install @cinacoin/erc6492
```

## Usage

```typescript
import { isERC6492Signature, encodeValidation, decodeValidation, validateSignature } from '@cinacoin/erc6492'

// Check if a signature is ERC-6492 encoded
const is6492 = isERC6492Signature(signature) // boolean

// Validate a signature (works for both EOA and ERC-6492)
const result = await validateSignature({
  address: '0x...',
  message: '0x...',
  signature: '0x...',
  provider: provider,
})

console.log('Valid:', result.valid)
```

## Encoding/Decoding

```typescript
import { encodeValidation, decodeValidation } from '@cinacoin/erc6492'

// Encode a contract wallet signature
const encoded = encodeValidation({
  factory: '0x...',
  factoryCalldata: '0x...',
  signature: '0x...',
})

// Decode an ERC-6492 signature
const decoded = decodeValidation(encoded)
console.log('Factory:', decoded.factory)
console.log('Signature:', decoded.signature)
```

## Types

```typescript
interface ERC6492Signature {
  factory: string
  factoryCalldata: string
  signature: string
}

interface ValidationResult {
  valid: boolean
  reason?: string
}

interface ValidationInput {
  address: string
  message: string
  signature: string
  provider: any
}
```
