📋 How It Works

🔗 SIWE (Sign-In With Ethereum)

1. Connect — Connect your wallet via eth_requestAccounts.

2. Sign — Sign a SIWE message (EIP-4361) via personal_sign.

3. Verify — Verify the signature matches the address in the message.

🔑 Passkey (WebAuthn)

1. Register — Create a passkey via navigator.credentials.create().

2. Login — Authenticate via navigator.credentials.get().

3. Verify — Session persisted in localStorage with 24h expiry.

// SIWE — Sign-In With Ethereum
