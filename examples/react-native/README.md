# Cinacoin React Native Example

> React Native 移动端示例，展示钱包连接、Swap、多链等核心功能。

## 运行

```bash
npm install

# iOS
cd ios && pod install && cd ..
npx react-native run-ios

# Android
npx react-native run-android
```

## 功能

- **ConnectScreen** — 连接页面，含钱包列表和连接状态
- **SwapScreen** — Swap 交易页面
- **MultiChainScreen** — 多链资产总览
- **WalletList** — 可复用钱包列表组件

## 依赖

- React Native 0.76+
- React 18+
- Cinacoin React Native SDK
