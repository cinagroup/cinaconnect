# Cosmos Adapter

> `@cinaconnect/adapter-cosmos` — Cosmos blockchain adapter for CinaConnect.

## Installation

```bash
npm install @cinaconnect/adapter-cosmos
```

## Overview

Connect to Cosmos ecosystem chains (Cosmos Hub, Osmosis, etc.) using the CosmJS library. Supports Keplr, Leap, and other Cosmos wallets.
