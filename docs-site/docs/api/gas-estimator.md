# Gas Estimator

> `@cinaconnect/gas-estimator` — Gas estimation utilities for CinaConnect.

## Installation

```bash
npm install @cinaconnect/gas-estimator
```

## Overview

Estimate gas costs for transactions across multiple chains, with support for EIP-1559 and legacy gas models.
