# Starknet Adapter

> `@cinaconnect/adapter-starknet` — Starknet L2 adapter for CinaConnect.

## Installation

```bash
npm install @cinaconnect/adapter-starknet
```

## Overview

Enable Starknet wallet connections using Starknet.js. Supports Argent X, Braavos, and other Starknet wallets.
