# Custom Connectors

> `@cinaconnect/custom-connectors` — Custom wallet connector framework for CinaConnect.

## Installation

```bash
npm install @cinaconnect/custom-connectors
```

## Overview

Build custom connectors for wallets not supported out of the box. Define injection patterns, communication protocols, and UI flows.
