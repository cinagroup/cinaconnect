# XRPL Adapter

> `@cinaconnect/adapter-xrpl` — XRP Ledger adapter for CinaConnect.

## Installation

```bash
npm install @cinaconnect/adapter-xrpl
```

## Overview

Enable XRP Ledger interactions including XRP transfers, token operations (issued currencies), and NFT interactions.
