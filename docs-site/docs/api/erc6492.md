# ERC-6492

> `@cinacoin/erc6492` — ERC-6492 signature validation for contract wallets.

## Installation

```bash
npm install @cinacoin/erc6492
```

## Overview

ERC-6492 defines a signature validation format for contract wallets (smart accounts) that haven't been deployed yet.
