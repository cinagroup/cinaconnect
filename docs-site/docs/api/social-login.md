# Social Login

> `@cinaconnect/social-login` — Social login integration for CinaConnect.

## Installation

```bash
npm install @cinaconnect/social-login
```

## Overview

Allow users to connect using existing social accounts (Google, Apple, Twitter/X, Discord) with underlying wallet creation.

## Providers

- Google
- Apple
- Twitter / X
- Discord
- GitHub
