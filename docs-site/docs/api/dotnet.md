# .NET C#

> `@cinacoin/dotnet` — .NET SDK for Cinacoin.

## Installation

```bash
dotnet add package Cinacoin
```

## Usage

```csharp
using Cinacoin;

var config = new CinacoinConfig { ProjectId = "your-project-id" };
var cinaConnect = new CinacoinClient(config);
```
