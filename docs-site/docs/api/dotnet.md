# .NET C#

> `@cinaconnect/dotnet` — .NET SDK for CinaConnect.

## Installation

```bash
dotnet add package CinaConnect
```

## Usage

```csharp
using CinaConnect;

var config = new CinaConnectConfig { ProjectId = "your-project-id" };
var cinaConnect = new CinaConnectClient(config);
```
