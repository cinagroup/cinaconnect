# CinaConnect UI Theme

> `@cinaconnect/cinaconnect-ui-theme` — Default theme presets for CinaConnect.

## Installation

```bash
npm install @cinaconnect/cinaconnect-ui-theme
```

## Presets

- `default` — Clean, modern default
- `dark` — Dark mode optimized
- `minimal` — Minimalist style
- `gaming` — Gaming aesthetic
