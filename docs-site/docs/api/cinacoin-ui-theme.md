# Cinacoin UI Theme

> `@cinacoin/cinacoin-ui-theme` — Default theme presets for Cinacoin.

## Installation

```bash
npm install @cinacoin/cinacoin-ui-theme
```

## Presets

- `default` — Clean, modern default
- `dark` — Dark mode optimized
- `minimal` — Minimalist style
- `gaming` — Gaming aesthetic
