# Blockchain API

> `@cinaconnect/blockchain-api` — Unified blockchain data API for CinaConnect.

## Installation

```bash
npm install @cinaconnect/blockchain-api
```

## Overview

A unified interface for querying blockchain data across multiple chains, abstracting away chain-specific differences.
