# Wallet Buttons

> `@cinaconnect/wallet-buttons` — Pre-built wallet connection button components.

## Installation

```bash
npm install @cinaconnect/wallet-buttons @cinaconnect/core-ui
```

## Usage

```html
<cina-wallet-buttons
  wallets="metamask,coinbase,rainbow"
  mode="grid"
/>
```
