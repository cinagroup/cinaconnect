# iOS Swift

> `@cinacoin/ios-swift` — iOS native SDK for Cinacoin.

## Installation

```ruby
# Podfile
pod 'Cinacoin', '~> 1.0'
```

## Usage

```swift
import Cinacoin

let config = CinacoinConfig(projectId: "your-project-id")
let cinaConnect = Cinacoin.shared(config: config)
```
