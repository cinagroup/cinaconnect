# iOS Swift

> `@cinaconnect/ios-swift` — iOS native SDK for CinaConnect.

## Installation

```ruby
# Podfile
pod 'CinaConnect', '~> 1.0'
```

## Usage

```swift
import CinaConnect

let config = CinaConnectConfig(projectId: "your-project-id")
let cinaConnect = CinaConnect.shared(config: config)
```
