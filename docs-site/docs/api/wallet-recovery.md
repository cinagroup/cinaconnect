# Wallet Recovery

> `@cinaconnect/wallet-recovery` — Wallet recovery mechanisms for CinaConnect.

## Installation

```bash
npm install @cinaconnect/wallet-recovery
```

## Overview

Provide multiple recovery paths for lost wallets: social recovery with guardians, seed phrase recovery, passkey recovery, and more.
