# Payment Flow

> `@cinacoin/payment-flow` — Payment flow management for Cinacoin.

## Installation

```bash
npm install @cinacoin/payment-flow
```

## Overview

Define and manage payment flows with support for swaps, on-ramps, transfers, and batch payments.
