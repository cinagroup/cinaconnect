# Config

> `@cinacoin/config` — Configuration management for Cinacoin.

## Installation

```bash
npm install @cinacoin/config
```

## Usage

```typescript
import { Config } from '@cinacoin/config'

const config = new Config({
  projectId: 'your-project-id',
  environment: 'production',
})
```
