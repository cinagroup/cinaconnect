# Config

> `@cinaconnect/config` — Configuration management for CinaConnect.

## Installation

```bash
npm install @cinaconnect/config
```

## Usage

```typescript
import { Config } from '@cinaconnect/config'

const config = new Config({
  projectId: 'your-project-id',
  environment: 'production',
})
```
