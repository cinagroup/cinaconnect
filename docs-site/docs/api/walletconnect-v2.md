# WalletConnect V2

> `@cinaconnect/walletconnect-v2` — WalletConnect V2 protocol integration for CinaConnect.

## Installation

```bash
npm install @cinaconnect/walletconnect-v2
```

## Overview

Integrate with WalletConnect V2 protocol for QR code and deep-link wallet connections, while maintaining self-hosted infrastructure.
