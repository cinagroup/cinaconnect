# Android Kotlin

> `@cinaconnect/android-kotlin` — Android native SDK for CinaConnect.

## Installation

```groovy
// build.gradle
implementation 'com.cinaconnect:sdk:1.0.0'
```

## Usage

```kotlin
val config = CinaConnectConfig(projectId = "your-project-id")
val cinaConnect = CinaConnect(context, config)
```
