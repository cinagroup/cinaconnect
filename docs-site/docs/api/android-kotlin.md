# Android Kotlin

> `@cinacoin/android-kotlin` — Android native SDK for Cinacoin.

## Installation

```groovy
// build.gradle
implementation 'com.cinacoin:sdk:1.0.0'
```

## Usage

```kotlin
val config = CinacoinConfig(projectId = "your-project-id")
val cinaConnect = Cinacoin(context, config)
```
