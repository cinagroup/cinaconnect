# Gas Sponsorship

> `@cinaconnect/gas-sponsorship` — Gas sponsorship management for CinaConnect.

## Installation

```bash
npm install @cinaconnect/gas-sponsorship
```

Manage gas sponsorship policies, track gas usage, and configure Paymaster rules for sponsored transactions.
