# ENS Resolver

> `@cinaconnect/ens-resolver` — ENS (Ethereum Name Service) resolver for CinaConnect.

## Installation

```bash
npm install @cinaconnect/ens-resolver
```

## Overview

Resolve and reverse-resolve ENS names, manage ENS records, and integrate ENS into wallet connections.
