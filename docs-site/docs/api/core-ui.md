# Core UI

> `@cinacoin/core-ui` — Framework-agnostic UI components built with Web Components (Lit).

## Installation

```bash
npm install @cinacoin/core-ui
```

## Usage

```html
<cina-connect-button></cina-connect-button>
<cina-connect-modal></cina-connect-modal>
<cina-account></cina-account>
```

## Components

- `<cina-connect-button>` — Connect wallet button
- `<cina-connect-modal>` — Wallet selection modal
- `<cina-account>` — Account display component
- `<cina-network-selector>` — Chain switcher
