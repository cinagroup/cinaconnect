# Nuxt

> `@cinaconnect/nuxt` — Nuxt 3 module for CinaConnect.

## Installation

```bash
npm install @cinaconnect/nuxt @cinaconnect/core-sdk
```

## Usage

```typescript
// nuxt.config.ts
export default defineNuxtConfig({
  modules: ['@cinaconnect/nuxt'],
})
```
