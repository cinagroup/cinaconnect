# Nuxt

> `@cinacoin/nuxt` — Nuxt 3 module for Cinacoin.

## Installation

```bash
npm install @cinacoin/nuxt @cinacoin/core-sdk
```

## Usage

```typescript
// nuxt.config.ts
export default defineNuxtConfig({
  modules: ['@cinacoin/nuxt'],
})
```
